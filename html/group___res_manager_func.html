ViStatus _VI_FUNC viClose ( ViObject  vi)

Закрытие сессии подключения.

Parameters
[in]viИдентификатор менеджера ресурсов или сессии соединения. Если передан идентификатор сессии, то она будет закрыта. если передан идентификатор менеджера ресурсов, будут закрыты все сессии, открытые в этом менеджере.
Returns
код ошибки
  • VI_SUCCESS
  • VI_ERROR_INV_OBJECT
  • VI_ERROR_CLOSING_FAILED

Создание новой сессии подключения в текущем менеджере ресурсов. Идентификатор менеджера должен быть извлечен функцией viOpenDefaultRM.

Parameters
[in]sesnИдентификатор менеджера ресурсов
[in]nameАдрес подключения
[in]modeОпределяет режим, в котором осуществляется подключение(не реализованно)
[in]timeoutТаймаут, по которому осуществляется блокировка соединения, если того требует режим доступа(не реализованно)
[out]viУказатель на идентификатор создаваемой сессии подключения
Returns
код ошибки
  • VI_SUCCESS
  • VI_ERROR_INV_OBJECT
  • VI_ERROR_INV_RSRC_NAME
  • VI_ERROR_RSRC_NFOUND
  • VI_ERROR_TMO
